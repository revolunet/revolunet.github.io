About

Apr 11th, 2013

revolunet is a web technologies developers team based in Paris, France.

We promote open source and agility and deliver high quality web applications.

Our key technologies are :

Python / Django / Flask : python powered backend frameworks

Sencha ExtJS : advanced javascript MVC framework for professional web applications

Sencha Touch : cross-platform javascript framework for mobile web applications

AngularJS : awesome javascript framework to power your web applications

Phonegap : package your web apps for the various app stores : iOS, Android, BackBerry…

You can email us at contact@revolunet.com or check our website : revolunet.com
